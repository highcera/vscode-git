Part 4, 5, 6 (프론트엔드) 자료는 어디서 확인하나요?

Part 4, 5, 6 자료는 채팅방 상단의 고정 메시지(Pinned Message)를 확인하거나, 채팅 내역을 위로 올려보면 이미지와 함께 파일(PART4_Web_Server.md 등)이 업로드되어 있습니다. Part 1, 2, 3을 먼저 구축한 뒤, 해당 파일들을 추가로 다운로드하여 진행하면 됩니다.

설치나 생성 과정이 너무 오래 걸리거나 멈춘 것 같아요. (무한 로딩)

Part 4, 5, 6은 규모가 커서 한꺼번에 요청하면 허용량을 초과하여 무한 로딩에 걸릴 수 있습니다. "순서대로 하나씩 진행해줘"라고 명령하여 단계별로 구축하는 것이 좋습니다. 만약 장시간 멈춰 있다면 프로그램을 재실행하거나 끊어서 요청하세요.

구축은 다 했는데 웹 화면(대시보드)은 어떻게 실행하나요?

구축 완료 후 flask_app.py 파일을 엽니다.

우측 상단의 재생 버튼(Run)을 누릅니다.

venv -> python을 선택합니다.

터미널 창에 Running on http://127.0.0.1:5000/ 같은 주소가 뜨면, Ctrl 키를 누른 채 해당 주소를 클릭하면 웹 브라우저에서 대시보드가 열립니다.

GPT API는 유료인가요? 결제 안 하면 못 쓰나요?

제미나이(Gemini)는 무료로 사용 가능하지만, GPT API는 유료 결제가 필요합니다. 결제를 원치 않으시면 Antigravity에게 "GPT 분석 기능은 제외하고 진행해줘"라고 요청하면 GPT 없이 시스템을 구축할 수 있습니다.

데이터 업데이트는 어떻게 하나요?

미국 주식 시장 마감 후 데이터가 갱신됩니다. 한국 시간으로 보통 화요일 오전에 "화요일 오전에 데이터가 업데이트되었으니 전체 업데이트해줘"라고 Antigravity에게 명령하면 최신 데이터로 분석 내용이 갱신됩니다.

Python(파이썬) 설치는 어떻게 하나요?

Python 공식 홈페이지에서 다운로드할 수 있습니다. 설치 시 "Add Python to PATH" 옵션을 반드시 체크하고 설치해야 명령어가 정상적으로 인식됩니다.

백엔드(Backend)와 프론트엔드(Frontend)가 뭔가요?

백엔드 (Part 1, 2, 3): 데이터 수집, 분석 등 눈에 보이지 않는 뒷단에서 작동하는 시스템입니다.

프론트엔드 (Part 4, 5, 6): 백엔드에서 분석한 데이터를 웹사이트 형태(대시보드, 그래프 등)로 시각화하여 사용자가 볼 수 있게 만드는 화면입니다.

마켓맵이나 AI 분석 결과가 화면에 안 나옵니다.

데이터 연결이 끊겼거나 분석이 완료되지 않았을 수 있습니다. Antigravity에게 "데이터 연결이 잘 되어 있는지 확인해줘" 또는 "다시 실행해줘"라고 요청해 보세요. 또한, 모델을 Gemini 3.0이나 Opus 4.5로 설정하여 진행하는 것을 권장합니다.

인코딩 에러(cp949, utf-8)가 발생합니다.

윈도우 환경에서 한글이나 이모지 처리 시 인코딩 문제가 발생할 수 있습니다. 에러 메시지를 Antigravity에게 그대로 보여주고 "이모지 문제나 인코딩 오류를 해결해서 코드를 수정해줘"라고 요청하면 자동으로 수정해 줍니다.

run_app.bat 파일은 무엇인가요?

프로그램을 쉽게 실행하기 위한 배치 파일입니다. 구축이 완료된 후 매번 코드를 열 필요 없이 이 파일을 실행하면 서버가 구동되도록 설정할 수 있습니다. Antigravity에게 실행 파일을 만들어달라고 요청할 수 있습니다.
