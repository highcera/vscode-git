1차시. 빠른 실행 도구 모음에 어떤 도구를 추가하면 좋을까요?
1) 열 너비 자동 맞춤 : 셀의 너브를 자동으로 맞춤
    전체 열 지정, 컬럼 오른쪽 경계에서 마우스 좌 더블 클릭, 자동 맞춤 > 너비가 긴 열의 행이 있으면 그 폭에 맞춤 > 폭을 적용할 행, 열에 위치 후 열너비 자동 맞춤
2) 카메라 : 원하는 윛에 자유롭게 배치
   결재란, PDF : 기존 서식에 넣으려면 간격을 맞추기 어려움 
   기타명령 | 리본 메뉴에 없는명령 | 카메라 
   영역 선택 후 단축키 > 놓을 위치에 놓음. 서식 > 그림 테두리 > 윤곽선 없음
   그림을 더블클릭 > 원본데이터로 이동 > 내용 변경 시 연동
3) 전체 병합 :: 행별로 각각 병합 시킬 수 있음. 기존 대비 유용
    홈 > 맞춤 > 병합하고 가운데 맞춤 > 전체 병합 > 오른쪽 마우스 > 빠른 실행 도구 모음에 추가
4) 값 붙여넣기 : 값만 붙여넣기, 모든 명령 > 값 (값 붙여넣기), copy & paste 위치 오른 마우스, V도 유사 속도
5) 텍스트 배분 : 셀에 텍스트를 균등하게 배분하여 조정
    모든 명령 > 텍스트 배분 추가

3차시. 데이터 입력 시 자동으로 순번이 적용되게 할 수 있나요?
1. ROW : 성명에 입력이 있으면 NO를 생성
             if(C6="",  "", row()-5)

2. COLUMN : ROW 만큼 많이 사용하지 않지만 vlookup 활용
    VLOOKUP($B$5,$H$5:$L$29,COLUMN()-1,0)\
    임직원 정보의 이름만 입력 : 데이터>데이터 유효성 검사>데이터 유효성 검사>설정>목록>원본

3. ROW, 조건부서식
성명 입력 시 테두리 자동 적용
NO는 ROW네서 적용한 자동 순번 적용
조건 적용할 범위지정
홈>조건부서식>새규칙>수식을 사용하여 서식을 지정할 셀 결정>규칙 설명 편집>=$C6<>"">서식>윤곽선>확인


4차시. 중복된 값을 입력하지 못하게 할 수 없나요?
ID 입력 영역 선택 > 데이터 > 데이터 유효성 검사 > 데이터 유효성 검사 > 설정 > 사용자 지정 > 수식 > =COUNTIF($B$5:$B$21,B10)=1 > 오류 메세지 > 제목 다시 입력해 주세요., 오류 메세지 ID가 중복되어 다시 입력을 해주셔야 합니다.

스타일 : 중지 > 입력불가, 경고 > 입력은 가능

3. 오류 메시지 스타일은 상황에 따라 중지, 경고, 정보 중 하나로 설정할 수 있다.

6차시. 근속년수와 같은 날짜 계산을 함수로 할 수 있을까요?
1. DATEDIF 함수를 사용하여 정확한 재직 기간을 손쉽게 계산할 수 있다.
   =DATEDIF($F9,$I$5,"Y")  =DATEDIF($F9,$I$5,"m")  =DATEDIF($F9,$I$5,"d")
2. & 연산자를 사용하여 재직 기간의 년, 월, 일을 하나의 셀에 보기 좋게 표시할 수 있다.
   =DATEDIF(F8,$G$5,"y")&"년 "&DATEDIF(F8,$G$5,"ym")&"개월 "&DATEDIF(F8,$G$5,"md")&"일"
3. EDATE 함수를 사용하여 특정 날짜로부터 몇 개원 후의 대금 지급일을 간편하게 계산할 수 있다.
4. EOMONTH 함수를 사용하여 대금 지급 달의 마지막 날을 정확히 계산할 수 있다.
      진행일        대금 지급 기한(개월)    대금 지급일	         말일 결제
   2030-05-20	           1                   =EDATE(E6,F6)      =EOMONTH(E6,F6)
